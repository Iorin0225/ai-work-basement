# AI Work Basement

このリポジトリは、Iorin0225がAI駆動作業を行うためのベーステンプレートです。コーディングだけでなく、調査・設計・ドキュメントなど幅広い仕事フローを支えるために構成されています。

## テンプレート構成

1. `docs/` 以下には、実際の成果物ではなく「このファイルにはこういう内容を書いてください」というテンプレートだけを置きます。各ファイル冒頭に `CHECKPOINT.md` のようなコメント付きの指示が記されているので、それをコピーして自分のプロジェクト用に記述を始めてください。
2. 実例・記入例を見たい場合は `samples/` を参照してください。`samples/demo/` にある README や各 docs ファイルはフォーマットを埋めたサンプルです。
3. `README.md` はテンプレートの使い方を伝えるための補足ドキュメントです。必要に応じて追記・拡張して運用に合わせてください。

## 運用メモ

1. ドキュメントを更新する前に `docs/` 以下のテンプレートをコピーし、コメントを参考に自分のプロジェクトの内容を記述してください。元のテンプレートはそのまま残しておくと次のセッションで再利用できます。
2. `docs/CHECKPOINT.md` のコメントブロックを例として、書くべき内容や守るべきルールが各ファイルに示されています。該当セクションをそのままコピーして使うと迷いが少なくなります。
3. 作業記録や検討のアウトラインは `samples/` を見ながら埋めていき、必要に応じて README に補足や外部リンクを追記してください。

## 持続的な使い方

1. 指示がクリアになったら `samples/` の該当例をベースに内容を埋めていき、必要な箇所だけ本物の `docs/` ファイルに記録する。
2. 各セッションの終わりに `docs/CHECKPOINT.md` のフォーマットを参考に状況・次の予定・意図を記録しておくと、次回以降の着手がスムーズです。
3. テンプレートが足りないと感じたら `docs/` に新しいファイルを追加し、その中にコメント形式で指示を残しておくと他の参加者も同じ形式で書けます。
