# AI Work Basement

このリポジトリは、Iorin0225がいわゆるAI駆動作業をするための基礎となるテンプレートです。
コーディングに限らずあらゆるお仕事に適応可能なことを目指して制作しています。

## つかいかた

このリポジトリはテンプレートなので、forkしてお使いください。

### docs

docs ドキュメントの中にベースとなるドキュメント群があります。
このリポジトリではそれらをベースに仕事を進めていきます。

- PLAN.md: 何をしたいのか。
- SPEC.md: 具体的な作業の試用をまとめる。
- TODO.md: タスクをストック・管理する
- KNOWLEDGEMENT.md: 必要な背景知識。
- CHECKPOINT.md: 各Session終了時に書き残すレポート群。

### スタート

docsを書き始めましょう。それらもAIと会話しながら書き進めるのを推奨します。
