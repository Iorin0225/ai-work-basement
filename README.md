# AI Work Basement

このリポジトリは、AIを使った業務ドキュメントを手早く立ち上げるためのテンプレートです。まず fork し、仕事ごとに README や docs を自分のリポジトリに展開してください。

## テンプレートの役割

- `docs/` には個別成果ではなく、書き方の指示をコメント付きで残したテンプレートを置いています。
- `samples/` は記入済みの例です。構造を眺めて、自分の内容を埋めてください。

## 手順

1. `docs/` のテンプレートをコピーして fork 先に貼り付け、プロジェクト用の内容を書き込みます。
2. `CHECKPOINT.md` の例を見ながら、「何をしたか」「次に何をするか」「その意図」を記録します。

## 継続的な使い方

1. セッション終了時には `docs/CHECKPOINT.md` を使って短く振り返りを残します。
2. テンプレートが足りないときは `docs/` に新しいファイルを置き、コメントで書き方を示します。

## 補足

- 複数業務を同一リポジトリで扱う必要が出た場合にだけ、`docs/` 内で別ディレクトリを作って構成を複製してください。普段は fork 先で個別に運用します。
